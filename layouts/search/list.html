Site Search

The search bar in the header can be used to look for one or more case-insensitive strings contained in the content body of this site, space delineated. See the examples below for more advanced search queries.
  • Fuzzyness: foo~1 returns articles containing "foo" and permutations of "foo" within a one-char difference (e.g. "foz")
  • Wildcards: bar* returns articles containing "bar" and any subsequent character(s)
  • Title tags: title:baz returns articles containing "baz" in their titles
  • Operators: +quux -xyzzy returns articles containing "quux" but not "xyzzy"
  • Weights: corge^10 grault^2 returns articles containing "corge" and/or "grault" but "corge" will have 5x relevance
